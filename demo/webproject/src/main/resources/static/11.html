数据级别缓存：

  • 在数据模型上加@Cache注解，参数：rcCache[默认fasle]，lcCache[默认false]，rcETime[默认1200秒]，则MysqlTemplate的save，update，delete，get方法会进行缓存操作
  • 数据模型DEMO地址：http://127.0.0.1:8888/lesson11/test
  • 在mapper xml 配置里加上对应缓存参数,如rcCache=true，则MysqlTemplated的getList，execute方法会进行缓存操作
  • mapper配置的DEMO地址：http://127.0.0.1:8888/lesson11/test2
  • 支持防缓存穿透，防缓存击穿，防缓存雪崩，详细介绍见/file/缓存穿透、缓存击穿、缓存雪崩区别和解决方案.docx
